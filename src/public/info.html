IdF NRW - LAGEskizze

Das IdF NRW - LAGEskizze ist ein Werkzeug zur Lagedarstellung auf Kartenbasis. Es basiert auf dem Lagedarstellungssystem NRW, dort der zentralen Lageskizze, und bietet die Daten des NRW-Atlas als Grundlage.

Das IdF NRW-LAGEskizze ist als OpenSource von den Feuerwehren in NRW kostenfrei nutzbar (siehe Lizenz) und wird in der vorliegenden Version in einer Kooperation zwischen dem IdF NRW und der EFTAS Fernerkundung Technologietransfer GmbH bereit gestellt.

Software: IdF NRW LAGEskizze (Stab)

Digitale Lagedarstellung für operativ-taktische Stabsarbeit.

LAGEskizze ist die Adaption der zentralen Lagedarstellung eines Unfalls, eines Feuers oder einer Katastrophe, wie sie im Lagedarstellungssystem Nordrhein-Westfalen (NRW) vom Institut der Feuerwehr NRW vorgeschlagen wird. Sie fügt sich thematisch in die analogen Schadenskonten, Bereitstellungsräume, Einsatzabschnitte etc. der Taktischen Arbeitswand ein und lässt sich über eine Splitscreen-Variante unabhängig von der eigentlichen Lagedarstellung bedienen.

Es handelt sich um ein Javascript-basiertes System zur raumbezogenen Visualisierung, das eine Unterstützung bei der konsistenten und strukturierten Einsatzstellen-Organisation in Einsätzen der Feuerwehr zum Ziel hat. Es implementiert die Vorgaben der Feuerwehr-Dienstvorschrift 100.

Die Open Source Anwendung ist unter Nutzung von node.js und AngularJS geschrieben. Der Quellcode ist eingestellt auf GitHub unter https://github.com/eftas/LAGEskizze

Für die Nutzung auf dem Desktop etwa in Einsatzleitwagen oder dem Stabsraum ist die Anwendung in einen einfach zu nutzenden „Electron-Container“ gekapselt worden. Die jeweils aktuellste Version kann über die EFTAS-Homepage herunter geladen werden.

Als Geobasisdaten (ATKIS / ALKIS) nutzt die aktuelle Version das Open Data-Angebot des NRW-Atlasses, der vom Land Nordrhein-Westfalen (Geobasis.nrw) bereit gestellt wird. Das entsprechende Copyright liegt beim Bundesland NRW, die jeweiligen Nutzungsbedingungen sind zu beachten. Bezüglich der Taktischen Zeichen wurde auf Konformität zur "Empfehlung für Taktische Zeichen im Bevölkerungsschutz" (Ständige Konferenz für Katastrophenvorsorge im Bevölkerungsschutz, 2010) geachtet.

Installationsanleitung

Wichtiger Hinweis: Mit der Installation der Software stimmt der Nutzer oder die Nutzerin den Lizenzbedingungen und den im Disclaimer gegebenen Hinweisen zu. Er oder sie erkennt die Urheberschaft der Software ebenso an, wie diejenigen der Geodaten und der taktischen Zeichen.

Das Archiv LAGEskizze-win32-x64.zip enthält alle zum Betrieb der Software notwendigen Komponenten inkl. der unter Lizenz genannten Komponenten, der Taktischen Zeichen und der Links zu den Geodaten des NRW-Atlas.

Das Archiv ist lediglich in ein Verzeichnis zu entpacken. Ein Doppelklick auf die ausführbare Datei LAGEskizze.exe startet die Software. Es wird empfohlen, sich auf dem Desktop eine Verknüpfung zur Datei LAGEskizze.exe anzulegen. Die Datei darf nicht aus dem Verzeichnis – etwa auf den Desktop - verschoben werden!
